# Co-WIN Portal Clone - Mini Project

A high-fidelity clone of the Indian Government's **Co-WIN (Winning Over COVID-19)** registration portal, built using **Next.js**, **TypeScript**, and **Bootstrap**. This project focuses on a responsive UI/UX that closely matches the official vaccination platform.

## ‚ú® Features

* **Fixed Navigation Bar**: Includes the Ministry of Health branding, language switcher, and accessibility controls.
* **Hero Section**: Features the "200 Crore Vaccinations" milestone badge and the interactive India vaccination map.
* **Floating Support Button**: A persistent "Support for COVID-19" side button that stays on screen during scrolling.
* **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
* **Next.js Routing**: Seamless navigation between pages like Dashboard, ABHA, and FAQ using the Link component.
